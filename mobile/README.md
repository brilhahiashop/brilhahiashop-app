BRILHAH IA SHOP - Mobile (Expo)
=================================
Quick start (local):
1. Install Node.js and Expo CLI / EAS CLI (recommended)
   npm install -g expo-cli eas-cli
2. In mobile/ run:
   npm install
   npm start
3. To build APK with EAS (expo.dev):
   eas build -p android --profile production
Notes:
- Set BACKEND_URL in app.json extra to your backend URL (Render or other)
